## 📌 Overview

This project presents an interactive Finance Performance Dashboard built using Microsoft Excel to analyze revenue, expenses, profit, and budget variance.

The dashboard enables users to track financial performance, monitor cost distribution, and identify areas of overspending.

---

## 🎯 Key Metrics

* Total Revenue: ₹3.75Cr
* Total Expense: ₹3.85Cr
* Net Profit: -₹10.1L (Loss)
* Budget Variance: ₹2.72Cr

---

## 🔍 Key Insights

* Expenses exceed revenue, resulting in overall loss
* Sales and Operations departments contribute the highest expenses
* Budget vs actual analysis shows overspending across departments
* Monthly revenue fluctuates, indicating unstable financial performance

---

## 📊 Features

* Interactive slicers (Department, Payment Mode, Date)
* Monthly revenue trend analysis
* Expense breakdown by department
* Budget vs actual comparison
* Payment method analysis
